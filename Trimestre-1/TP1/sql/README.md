# TP1 — Visualização de Dados com Google Looker Studio

## Descrição
Primeiro contato com visualização de dados, conectando fontes externas
ao Google Looker Studio e criando dashboards interativos.

## Ferramentas Utilizadas
- Google Looker Studio
- Google Sheets
- File Upload (conector CSV)

## O que foi feito

### Exercício 1 — Hello World
- Importação de um arquivo CSV externo via conector File Upload
- Criação do primeiro relatório com visualização em tabela simples

### Exercício 2 — Coffee Shop Dashboard
- Importação e limpeza de dados da cafeteria Hermann's Cake and Coffee Shop
- Organização dos dados em formato tabular no Google Sheets
- Criação de dashboard com:
  - Tabela completa dos dados
  - Gráfico de pizza por Product Name
  - Gráfico de pizza por Product Type
  - Visão Geral (KPI) de unidades vendidas (Number Sold)

## Aprendizados
- Conexão de fontes de dados externas ao Looker Studio
- Limpeza básica de dados no Google Sheets
- Criação de visualizações: tabelas, gráficos de pizza e KPIs
