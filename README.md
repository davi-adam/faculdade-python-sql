# Exercícios Python & SQL — Segundo Semestre

Repositório com exercícios selecionados das disciplinas de **Programação com Python**, **Python para Dados**, **Visualização de Dados e Introdução a SQL** e **Modelagem Relacional com SQL**
cursadas ao longo do segundo semestre na faculdade.

## Objetivo

Registrar o progresso e o aprendizado prático adquirido durante o semestre,
servindo como portfólio técnico e referência pessoal de estudo.

Os exercícios foram escolhidos por representarem bem os conceitos trabalhados
em cada etapa — não são todos, mas os que melhor demonstram evolução e domínio
dos temas abordados.

## Tecnologias

- **Python 3** — lógica, estruturas de dados, funções, arquivos e bibliotecas
- **SQL / SQLite** — modelagem relacional, DDL, DML e consultas
- **Pandas** — manipulação e análise de dados
- **BeautifulSoup** — web scraping
- **SQLAlchemy** — integração Python com bancos relacionais
- **Google Looker Studio** — visualização de dados (TP1 e TP2 do Trimestre 1)

## Organização dos arquivos

Cada arquivo segue o padrão `exNN_nome_descritivo.py` ou `.sql`.
O enunciado do exercício está comentado no topo de cada arquivo,
seguido da solução desenvolvida.
